This is test project for Technogies Pvt Ltd. 

Technogies:

- Eclispse IDE Version: 2019-03 (4.11.0).
- Java console application.
- Junit 5 for test cases.

Step to Run:

- Import project as a General -> Existing projects -> Finish
- Add Junit5
	- Right click on project and select Build Path.
	- Then select Cofigure Build Path.
	- Add Library -> Select Junit.
	
- Build and Run project.

Step to Run Test Cases:

- Run as JUnit Test, It will run all prepared test cases.
